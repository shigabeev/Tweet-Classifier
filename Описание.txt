# Тематический анализ.
Конечная цель — создать ПО, которое на основе данных из социальной сети узнает интересы выбранных пользователей, которые можно будет использовать в целях рекламы.
Для данных целей будет применятся анализ натурального языка, в частности, Наивный Байесовский классификатор. Выбираются интересующие тематики, я выбрал следующие: технологии, новости, жизнь, любовь и юмор. По данным темам были найдены тематические страницы, соответственно: Хабрахабр, Медуза, WhySoDisgusting, Samanta, Анатолий Ноготочки. Из каждой страницы было собрано по 200 твитов(get_tweets.py). 20 из которых будут добавлены в тестовую выборку.
Далее, собирается статистика частоты использования каждого слова (freq.py) и вычисляются все необходимые вероятности(пока что в excel, Freqs/F.csv). После этого используется GaussianNB из пакета sklearn, который предоставляет долгожданный результат.
